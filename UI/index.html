This application is a front-end for the listener module implemented on a Raspberry Pi 4. This frontend recieves data from the listener for display. This includes information about recent classifications, device status, and detected gunshot events.

View the source code →
Technical Approach

The listener module makes use of a small microphone to continually record ambient noise. Recordings are chunked to be 4 seconds long. When audio activity is detected in a 4-second chunk, the audio chunk is preprocessed through a short-time fourier transform, which of the squared magnitude is taken, resulting in an spectrogram.

The spectrogram is a representation of audio that expresses the signal characteristics in terms of frequency over time. The spectrogram is used as the input to a MobileNetV3 CNN that then classifies the spectrogram into one of ten categories.
